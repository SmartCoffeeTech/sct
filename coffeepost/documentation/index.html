About

Nostalgia is a HTML responsive landing page - personal or corporate, based on full-screen slider. The template is maintained in a minimalist, contrasting style. Background images kept in retro tones contrast with modern minimalist content. Landing page is divided into sections: hero, sliding gallery, features with video preview, testimonials and contact with map of location and working contact form. Template has a responsive layout that looks great on mobile and tablet devices.

Template Features

  • jQuery Powered
  • Responsive Design
  • Fullscreen Slideshow
  • Working Contact Form
  • Working Newsletter Form
  • Build-in Lightbox
  • Font Face Fonts
  • Valid XHTML Code
  • PSD Source Included
  • Documentation Included

Plugins

Contact form

This project includes a contact form ready for use.
All you need is to configure some constants located in file plugin/contact-form/config.php:

Required data

Constant Description
CONTACT_FORM_TO_NAME name of recipient eg. "My Company"
CONTACT_FORM_TO_EMAIL e-mail address of recipent eg. "mycompany@gmail.com"

Optional SMTP settings

Constant Description
CONTACT_FORM_SMTP_PORT your SMTP port eg. "587"
CONTACT_FORM_SMTP_HOST your SMTP host eg. "smtp.gmail.com"
CONTACT_FORM_SMTP_USER login to your e-mail account eg. "mycompany"
CONTACT_FORM_SMTP_PASSWORD password to your e-mail account
CONTACT_FORM_SMTP_SECURE secure connection protocol: "SSL" or "TLS"

Texts and messages

Constant Description
CONTACT_FORM_SUBJECT subject of e-mail message
CONTACT_FORM_MSG_INVALID_DATA_NAME text displayed when name is empty
CONTACT_FORM_MSG_INVALID_DATA_MAIL text displayed when e-mail address is invalid
CONTACT_FORM_MSG_INVALID_DATA_MESSAGE text displayed when message field is empty
CONTACT_FORM_SEND_MSG_OK text displayed when e-mail was sent successfully
CONTACT_FORM_SEND_MSG_ERROR text displayed when e-mail cannot be sent

Newsletter form

This project includes a newsletter form ready for use.
All addresses are stored in /newsletter/newsletter.csv file by default. This file is protected by .htaccess file.
All you need is to configure some constants located in file plugin/newsletter-form/config.php:

Required data

Constant Description
NEWSLETTER_FORM_DATA_FILE_PATH path to file in which addresses have to be stored
NEWSLETTER_FORM_MSG_INVALID_DATA_MAIL text displayed when e-mail address is invalid
NEWSLETTER_FORM_MSG_FILE_ERROR text displayed when e-mail address could not be saved
NEWSLETTER_FORM_MSG_MAIL_EXIST text displayed when e-mail address exists
NEWSLETTER_FORM_SEND_MSG_OK text displayed when e-mail address has been added

FAQ

Installing template

To install this template upload all files from www folder into HTTP server.

Twitter settings

Open file /script/main.js and set two values: twitter.name (your Twitter nickname) and twitter.count (number of tweets which should be displayed) in options object (section: Template options).

Changing Google Maps coordinates

Open file /script/main.js and set value: googleMap.coordinates in options object (section: Template options).

Changing background images

Open file /script/main.js and set value: supersized.slide in options object (section: Template options).
